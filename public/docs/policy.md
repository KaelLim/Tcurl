# 慈濟短網址服務 TCurl 使用說明

**發佈單位**：資訊處
**發佈日期**：2025 年 1 月

---

## 關於 TCurl

為提升本會數位內容的一致性與管理效率，資訊處已建置內部短網址服務平台 **TCurl**（url.tzuchi.org），提供同仁便利的短網址生成與管理功能。

---

## 為什麼要使用內部短網址？

過去同仁常使用 reurl.cc、ppt.cc、bit.ly 等外部服務，雖然方便，但存在以下考量：

| 外部服務的限制 | TCurl 的優勢 |
|---------------|-------------|
| 網址無法呈現慈濟品牌 | 使用 `url.tzuchi.org` 專屬網域 |
| 點擊數據由第三方掌控 | 資料完全由本會管理 |
| 服務可能停止或連結失效 | 永久維護，連結穩定 |
| 無法深入分析成效 | 完整的點擊統計與分析 |
| 外觀樣式無法客製 | QR Code 可加入慈濟 Logo |

---

## 平台功能

- **短網址生成**：將長網址轉為簡短易記的連結
- **自訂短代碼**：可自訂有意義的代碼（如 `url.tzuchi.org/s/2025gala`）
- **QR Code 生成**：支援客製化顏色、樣式，可加入慈濟 Logo
- **數據分析**：追蹤點擊次數、區分連結點擊與 QR 掃描
- **進階設定**：支援密碼保護、過期時間設定

---

## 使用準則

### 建議使用 TCurl 的情境

- 官方社群媒體貼文（Facebook、LINE、Instagram）
- 電子報、EDM、活動通知
- 印刷品（海報、傳單、活動文宣）
- 對外新聞稿、公告
- 任何代表慈濟基金會發佈的數位內容

### 使用方式

1. 前往 https://url.tzuchi.org
2. 使用 Google 帳號登入（@tzuchi.org 或 @tzuchi-org.tw）
3. 輸入原始網址，即可生成短網址與 QR Code

## 常見問題

**Q：我已經用外部短網址發佈了，需要改嗎？**
A：已發佈的內容不需更動，未來新發佈的內容建議改用 TCurl。

**Q：外部短網址還能用嗎？**
A：個人用途不限制，但代表本會官方發佈時，建議使用 TCurl 以維持品牌一致性。

**Q：遇到問題怎麼辦？**
A：請聯繫資訊處 service@tzuchi.org.tw，或參考線上說明 https://url.tzuchi.org/docs
